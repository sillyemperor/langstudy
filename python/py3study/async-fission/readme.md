# 客户端

- 生成一个唯一ID，整型数[1,n]
- 启动以后用ID向服务的注册自己
- 收到消息后裂变出两个消息，发送给随机的两个客户端（随机ID，避开接收端和自己）

# 服务端

- 接受客户端注册，为每一个ID建立连接
- 接收转发客户端的消息

# 消息体

- 注册消息
```json
{
  "client": "<客户端ID>"
}
```

- 裂变消息
``` json
{
  "id":"<消息ID>",
  "from":"<发送者ID>",
  "to":"<接收者ID>",
  "depth":<裂变深度，每次裂变减1，为0停止裂变>,
  "send_ts":<发送时间戳>,
  "receive_ts":<接收时间戳>
}
```
